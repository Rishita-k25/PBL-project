📦 Implemented DSA Concepts

Stack (LIFO) — Application history & undo
Queue (FIFO) — Application processing order
Linear Search — Find jobs by keyword
Binary Search — Find jobs by exact salary
Bubble Sort — Sort jobs by salary
Selection Sort — Sort jobs alphabetically
